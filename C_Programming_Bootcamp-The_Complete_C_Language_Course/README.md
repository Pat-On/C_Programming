# Useful commands

```bash
gcc filename.c
```

```bash
gcc hello.c -o hello
```

```bash
gcc filename.c -o program -Wall -Wextra -g
```

- Wall — show all warnings
- Wextra — extra warnings
- g — include debugging info
- O2 — optimize code (faster execution)
- std=c11 — use C11 standard


```bash
gcc file1.c file2.c file3.c -o myprogram
```
