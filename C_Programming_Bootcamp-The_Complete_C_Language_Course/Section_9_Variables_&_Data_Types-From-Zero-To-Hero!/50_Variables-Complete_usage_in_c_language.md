1. Variable creation
2. structure in c language `<typo> <name>;`

   2. `double temp;`

## assignment 

1. Put some value in a Variable
2. Structure in C Language

```c

double temp;

```

## Reading Input from User

1. Reading data from the user
2. Structure in C Language (using Assignment)

```c
int grade1;
int grade2;
grade1 = 80;
grade2 = 100;
print("Average = %d\n", (grade1+grade2)/2);
```


## Reading input from User

```c
int grade1;
int grade2;

scanf("%d", &grade1);
scanf("%d", &grade2);


print("Average = %d\n", (grade1+grade2)/2);

```
