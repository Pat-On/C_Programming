# Weather station

1. Write a program that gets from the user a temperature ("double") in Calcius Degrees

The program should calculate and convert the temperature from Celcius Degrees to Fahrenheit Degrees and print it

```
T(F) = T(C) * 1.8 + 32
```

2. Write a program that gets from the user a temperature ("double") in Fahrenheit Degrees

The program should calculate and convert the temperature from Fahrenheit Degrees to Celcius Degrees and print it


```
T(C) = (T(F) - 32) / 1.8
```
